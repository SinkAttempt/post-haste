# Post Haste — Tech (Current)

**Phase:** Web prototype (validate core loop)
**Stack:** HTML5 Canvas / Vanilla JS
**Target:** Mobile browser, portrait mode, touch input
**Future:** Unity (if loop validates and art studio engaged)

## Prototype Scope

### Core Loop (target)
- Virtual joystick movement (one thumb)
- Auto-proximity interactions at stations
- Visual stack on character (mail/parcels pile up)
- Sorting desk: swipe-to-sort mechanic (swipe items to correct bins)
- Customer counter: auto-serve on proximity
- Money collection and basic upgrades

### Screen Flow (target)
- Menu -> Playing -> Day Results -> Upgrade Shop -> Next Day
- Day-based progression (each day = session)
- localStorage persistence (money, upgrades, day progress)

## NOT in prototype
- Staff hiring / automation
- Vehicle fleet
- International mail / customs
- Monetisation (ad integration)
- Accounts / cloud saves
- Sound / music
- Polish art (placeholder shapes with colour)
- Tutorial overlay
